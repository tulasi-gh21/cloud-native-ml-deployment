# Cloud Native ML Deployment using Docker, Kubernetes & Terraform

## Project Overview
This project demonstrates how to build and deploy a Machine Learning application using modern cloud-native technologies.

The project includes:
- Machine Learning model training
- FastAPI backend API
- Docker containerization
- Kubernetes deployment
- Terraform infrastructure folder

---

## Technologies Used

- Python
- FastAPI
- Scikit-learn
- Docker
- Kubernetes
- Terraform
- GitHub

---

## Machine Learning Model

The model is trained using the Iris Dataset.

Algorithm used:
- Random Forest Classifier

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
